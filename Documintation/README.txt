# ANS Management System

## Overview
**ANS Management System** is a Java desktop application built using **Java Swing** and connected to a **SQLite** database. The system aims to manage school data efficiently by allowing administrators to register, view, and remove students and teachers. It also includes statistical views such as top 10 students and counts of registered individuals.

## Features
- Register new **students** and **teachers**
- View and delete existing students and teachers
- Display total count of students and teachers
- View detailed information of each individual
- Display the **Top 10 students** based on performance or criteria
- GUI built with **Java Swing**
- Persistent data storage using **SQLite**

## Technologies Used
- **Java**
- **Swing (GUI)**
- **SQLite (Database)**

# نظام إدارة ANS

## نظرة عامة
**نظام إدارة ANS** هو تطبيق مكتبي بلغة Java باستخدام مكتبة **Swing**، ويعتمد على قاعدة بيانات **SQLite** لتخزين البيانات. يهدف النظام إلى تسهيل إدارة بيانات المدرسة من خلال إمكانية تسجيل وعرض وحذف الطلاب والمعلمين، بالإضافة إلى عرض إحصائيات مثل أعلى 10 طلاب وعدد المسجلين.

## المميزات
- تسجيل **طلاب** و**معلمين** جدد
- عرض وحذف الطلاب والمعلمين الحاليين
- عرض إجمالي عدد الطلاب والمعلمين
- عرض معلومات مفصلة لكل طالب أو معلم
- عرض **أفضل 10 طلاب** بناءً على الأداء أو معايير محددة
- واجهة رسومية باستخدام **Swing**
- تخزين البيانات بشكل دائم باستخدام **SQLite**

## التقنيات المستخدمة
- **Java**
- **Swing (واجهة المستخدم الرسومية)**
- **SQLite (قاعدة البيانات)**
